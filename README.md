# To-Do-List

A modern, responsive, and fully-featured Todo App built with HTML, CSS, and JavaScript. 
This app allows you to add, edit, delete, and complete tasks, categorize them, track progress via a dashboard, and switch between light and dark mode. 
It also supports keyboard shortcuts, including pressing Enter to add/update a task.

Features
    Add, edit, and delete todos
    Mark todos as completed
    Filter todos by category: Work, Study, Personal
    Search todos by name
    Dashboard shows:
    Total tasks
    Completed tasks
    Overdue tasks
    Category-wise task count
    Progress bar showing completion percentage
    Dark mode toggle (theme saved in localStorage)
    Mobile responsive design
    Keyboard support:
    Press Enter in input fields to add/update a todo
    Overdue highlighting for past-due tasks
    Local storage persistence
